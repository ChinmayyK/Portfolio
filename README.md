## Portfolio

Personal portfolio for Chinmay Kudalkar, built with Next.js, React, Tailwind CSS v4, and Framer Motion.

## Development

Run the local development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` starts the dev server.
- `npm run lint` runs ESLint.
- `npm run build` creates a production build.

## Notes

- App icons and favicon branding come from `public/logo.png` and `src/app/icon.png`.
- Main styling tokens live in `src/app/globals.css`.
- Project system visualizations live in `src/components/system-diagram`.

## Build Check

```bash
npm run lint
npm run build
```
